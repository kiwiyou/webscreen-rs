# webscreen-rs

Web Screencast written in Rust.

## Usage

Clone this repository and run using cargo.

```bash
    cargo run --release
```

## Configuration

`Settings.toml`
- `bind`: An ip address to bind to.
- `interval`: Screen update interval in milliseconds.
